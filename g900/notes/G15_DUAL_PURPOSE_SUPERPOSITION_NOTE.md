# G15 Dual-Purpose Superposition Note

## Status
Working note

## Purpose

This note records the emerging interpretation that each host cube carries both a **subjective purpose** and an **objective purpose**, and that this dual role explains how witnessed sign-flips propagate into a geodesic of superposed positions.

This note is not using “superposition” in a vague or decorative way.
It is using it in a structural project-specific sense.

---

## Core claim

Each host cube has two simultaneous roles:

- a **subjective purpose**
- an **objective purpose**

The subjective purpose is local.
The objective purpose is relational.

Subjectively, the cube carries its own sign-closing transport law.
Objectively, the cube witnesses the return law required for propagation into the neighboring host.

Thus a cube is both:

- a participant in its own cycle
- a witness for the next cycle

This dual role is the basis of the propagated superposition chain.

---

## 1. Subjective purpose

A cube’s **subjective purpose** is what it does from its own local stance.

In the current G15 framework, that means:

- it hosts a local lawful walk
- one full cycle is sign-closing rather than identity-closing
- the return is:

\[
n_{15} = -n_0
\]

So from the inside, the cube’s primary task is to carry a full local transport law that returns to the same host site with reversed sheet side or reversed sided stance.

This is the cube’s own cycle logic.

---

## 2. Objective purpose

A cube’s **objective purpose** is what its sign-closed return means for the larger structure.

The sign-flipped return is not merely a dead-end local fact.
It becomes the witnessed bridge into the neighboring host.

So objectively, the cube’s role is:

- to witness the flip
- to provide the lawful handoff into the adjacent cube
- to support propagation of the transport chain across hosts

Thus the neighboring cube becomes the witness to the sign-closed return.

This is the sense in which the objective role is larger than the cube’s private cycle.

---

## 3. Why “superposition” is the right local word

In this project, “superposition” does not mean an undefined blur or a generic quantum analogy.

It means that one cube-state carries two simultaneous meanings:

1. **subjective meaning**
   - what the cube is doing in its own local cycle

2. **objective meaning**
   - what that same state witnesses for the neighboring cube or for the larger propagation chain

So a superposed state is a state with dual role, not a state with lost identity.

This can be written schematically as:

\[
\text{superposed cube state}
=
(\text{local stance},\ \text{witnessed relational role})
\]

That is the intended meaning here.

---

## 4. Sign-closure and witness propagation

The current locked cycle rule is:

\[
n_{15} = -n_0
\]

with doubled-cycle restoration:

\[
n_{30} = n_0
\]

But this alone does not yet explain propagation across a field of neighboring cubes.

The propagation step comes from witnessing.

A local sign-flipped return in one cube becomes the matched witness state in the neighboring cube.

So the transport is better read as:

\[
n \to -n \xrightarrow{\omega} n'
\]

where:

- \(-n\) is the sign-closed return in the current host
- \(\omega\) is the witness relation into the neighboring host
- \(n'\) is the corresponding realized stance in that neighboring host

This is the basic propagation unit.

---

## 5. Geodesic of superpositions

If the witnessed flip is propagated repeatedly, the result is not just a set of isolated sign-closing loops.

It becomes a chain:

\[
n_0 \to -n_0 \xrightarrow{\omega} n_1 \to -n_1 \xrightarrow{\omega} n_2 \to -n_2 \xrightarrow{\omega} \cdots
\]

This is what motivates the phrase:

> geodesic of superpositions

The phrase should be read carefully.

### Why geodesic
Because the propagation is not arbitrary.
Each step follows the lawful minimal continuation imposed by the host transport and its witness relation.

### Why superposition
Because each local state simultaneously plays:

- a subjective local cycle role
- an objective witness role in the larger chain

So the propagated object is a chain of dual-purpose states.

---

## 6. Dual-role principle

The clearest compact statement so far is:

> A cube is both participant and witness.

More explicitly:

> Every host cube has a subjective purpose and an objective purpose.

Subjective purpose:
- carry its own local sign-closing cycle

Objective purpose:
- witness the sign-closed return in a way that enables lawful continuation into the neighboring host

This should now be treated as a working principle of the current model.

---

## 7. Relation to G15 and G30

This dual-purpose interpretation sharpens the role split between G15 and G30.

### G15
G15 is the local sign-closing host cycle.

It tells what a cube does in its own frame:

\[
n_{15} = -n_0
\]

### G30
G30 is the doubled or paired completion that restores identity:

\[
n_{30} = n_0
\]

But in the dual-purpose reading, G30 is not merely “walk the same thing twice.”

It is the system-level completion of the sign-closing law, possibly through the witnessed coupling between neighboring hosts.

So a useful informal phrasing is:

- G15 = what the cube does for itself
- G30 = what the pair does for the system

This is not yet a final theorem, but it is a good working distinction.

---

## 8. Local versus propagated meaning

A single cube state should therefore be read on two levels.

### Local reading
What is its position, sidedness, and posture within its own host?

### Propagated reading
What is its witness function for the neighboring host and the continuing chain?

A state that is meaningful on both levels is a superposed state in the project sense.

So superposition is dual-purpose legibility.

---

## 9. Minimal formal sketch

A very light formalism would treat a cube state as:

\[
s = (x,w)
\]

where:

- \(x\) = local host stance
- \(w\) = witnessed role or neighbor-facing interpretation

Then the cycle law acts locally on \(x\), while propagation acts through \(w\).

A slightly more expanded version would be:

\[
s = (a,\varepsilon,\tau; w)
\]

where:

- \(a\) = ambient host site
- \(\varepsilon\) = sign or sheet side
- \(\tau\) = local posture or transport phase
- \(w\) = witness relation into the neighboring host

This is not yet the final formal model, but it captures the dual-purpose idea faithfully.

## 10. Plain-language formulation

Plainly said:

A cube has its own job and the system’s job.

Its own job is to complete its local sign-closing walk.
The system’s job is to witness that walk so the next cube can lawfully continue the chain.

That is why each cube is doing two things at once.

That is why the propagated chain can be described as a geodesic of superpositions.
