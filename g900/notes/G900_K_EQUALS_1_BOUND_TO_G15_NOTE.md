# G900 k = 1 Bound to G15
## A short note on treating the Petersen core as the first candidate k = 1 closed carrier

Date: 2026-03-29

---

## Purpose

This note records a conjectural placement move in the project:

> what happens if \(k = 1\) is bound to \(G_{15}\)?

This is not being recorded as a proved theorem.
It is being recorded as a disciplined conjectural interpretation.

The idea is not that the executable-knowledge note has already derived the Thalean quotient core.
The idea is that the quotient core may be the first place where the root condition of admissible distinction becomes fully closed and theorem-legible in image space.

That is a strong possibility worth naming.

---

## 1. The two ingredients

### 1.1 The root declaration

The declaration

\[
k = 1
\]

is the root condition of knowability.

In the project’s epistemic language, this means:

- the minimal admissible distinction
- the smallest executable epistemic step
- the boundary below which knowability is not constructed

This declaration is not originally a graph claim.
It is not originally a graph size.
It is not originally a claim about a specific finite carrier.

It is a root condition.

In the Thalean construction, the transport object descends through the quotient tower

\[
G_{60} \to G_{30} \to G_{15} \cong L(\mathrm{Petersen}).
\]

On this deepest quotient core, the transport-induced sector system yields the matrix \(M\), and the aa image

\[
Q := MM^T
\]

satisfies the exact law

\[
Q = A^3 + 2A^2 + 2I.
\]

This makes \(G_{15}\) the first place where the transport system becomes algebraically exact in a very strong sense.

## 2. The binding proposal

To bind \(k = 1\) to \(G_{15}\) is to make the following conjectural interpretive move:

> \(G_{15}\) is the first candidate minimal closed carrier on which admissible distinction becomes theorem-legible as aa.

This does **not** mean:

- that \(G_{15}\) is numerically equal to \(k = 1\)
- that 15 vertices literally are “one distinction”
- that the whole Thalean theorem is already derived from the epistemic root declaration

Those would be overclaims.

The intended meaning is narrower and stronger:

- \(k = 1\) gives the minimal admissible root
- the Thalean construction provides a lawful climb
- \(G_{15}\) is the first place where that climb closes into an exact theorem-grade aa image

That is the proposal.

This binding is attractive for several reasons.

Without the binding, \(G_{15}\) is the deepest quotient and the algebraic core of the Thalean paper.

With the binding, \(G_{15}\) becomes something more:

> the first epistemically closed carrier in the project.

That is a much stronger role.

### 3.2 It gives the first exact aa law a root relation

The law

\[
Q = A^3 + 2A^2 + 2I
\]

can then be read not only as a quotient-core graph identity, but as the first place where admissible distinction closes into a fully exact lens-visible image law.

That is very powerful.

### 3.3 It sharpens the bridge program

If the binding is viable, then the bridge ladder becomes cleaner:

- **Binary** = scalar prototype of exact lens-relative law
- **G15** = first minimal closed operator core
- **G900** = larger carrier sought as an extension or ascent from the same closure grammar

That is a compelling route.
