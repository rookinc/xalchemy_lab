# Binary Row Freeze

- Generator:
  \(I_{n+1}=2I_n+\epsilon_n\), with bounded correction

- Natural scales:
  boundary \(2^n\)
  bulk \(4^n\)

- Lenses:
  \(\rho_o(n)=I_n/2^n\)
  \(\rho_s(n)=I_n/4^n\)

- Images:
  \(\Phi_o(n)=\log \rho_o(n)\)
  \(\Phi_s(n)=\log \rho_s(n)\)

- Exact perspective law:
  \(\Phi_o(n)-\Phi_s(n)=n\log 2\)

- Derived consequence:
  if boundary density stabilizes, bulk frame acquires gradient \(\log 2\)

- Honest role:
  scalar prototype of exact lens-relative law
