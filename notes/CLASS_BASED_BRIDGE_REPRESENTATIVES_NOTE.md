# Class-Based Bridge Representatives Note

## Checkpoint

The bridge representatives are now class-based on the actual-edge side.

### global_return
Representative of the even return-type class:
- q0 -> q1 -> q0

### global_square
Representative of the even 4-cycle family:
- q0 -> q1 -> q6 -> q3 -> q0

### global_twist
Representative of the odd 4-cycle family:
- q0 -> q1 -> q5 -> q2 -> q0

## Meaning

The bridge no longer depends on placeholder symbolic roles alone.

It is now formulated by choosing actual representatives of:
- the return-type class
- the even square-type class
- the odd twist-type class

That is the stronger geometric formulation.
