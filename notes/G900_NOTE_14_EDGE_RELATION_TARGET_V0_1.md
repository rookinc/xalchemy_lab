# G900 Note 14 — Edge-Relation Target v0.1

## New target
Move from cell-label quotienting to edge-relation quotienting.

## Why
The weighted triangular prism is an adjacency object.
So the correct carrier-to-prism derivation may depend on how adjacent carrier cells interact, not on how isolated cells are labeled.

## Next question
When two adjacent carrier cells meet, what transport or relation class do they realize?

## Goal
Collapse carrier edge relations into three types:
- top-face edge class
- bottom-face edge class
- rung edge class
