# Multi-Controller Holonomy Probe Template

## Goal

Test whether chart-sensitive meaning accumulates across multiple controllers even when physical exit traces look similar.

## Compare

- loop_A: all controllers in chart A=0 mixed state
- loop_B: all controllers in chart A=1 mixed state
- loop_C: mixed chart assignment

## Inspect

For each loop:
- physical trace
- chart trace
- per-controller summaries

## What would support the holonomic reading

Support increases if:
- physical traces stay the same or nearly the same,
- while chart traces differ systematically across loop assignments.

## Strongest sentence

If multi-controller loops preserve physical traces while altering chart traces, then framed meaning is accumulating across the loop and not merely at a single controller.
