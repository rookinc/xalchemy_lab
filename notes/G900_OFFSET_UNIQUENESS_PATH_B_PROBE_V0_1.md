# G900 Offset Uniqueness Path B Probe v0.1

## Goal
Probe the stronger uniqueness route:

> any admissible centered prism law compatible with the locked package must have d = 5

This goes beyond excluding only currently visible named rivals.

## Locked package
An admissible rival must preserve:

1. shared scalar center:
   - I = 145

2. centered prism form:
   - (145 - d, 145, 145 + d)

3. mediator role:
   - the rung is the unique central mediator

4. coarse collapse law:
   - top + bottom = 290 = 2 * 145

5. midpoint / predecessor-shell architecture:
   - midpoint supplies center
   - predecessor shell supplies first branch scale

6. prism-level status:
   - the offset must belong to the centered prism layer,
   - not to a later parity refinement or unrelated carrier statistic

## Probe question
Under these constraints, is there any admissible meaning left for d other than:

- d = 5 ?

## Rival families to eliminate at admissible-law level
A rival law fails if it does any of the following:

- breaks exact reconstruction of (140,145,150)
- uses the wrong structural layer
- treats a shell label as a displacement law
- treats adjacency as scale
- introduces a free normalization parameter
- preserves arithmetic but loses midpoint/predecessor-shell meaning

## Success condition
Path B succeeds if every admissible rival law either:

- collapses to the same half-layer reading in disguise, or
- ceases to be admissible under the locked package.
