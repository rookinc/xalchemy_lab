# Theorem Preimage Comparison Note

## Checkpoint

The bridge signature
- (R,Q,W) = (0,0,1)

has at least two realized invariant preimages:
- (A,sigma,tau) = (0,0,1)
- (A,sigma,tau) = (1,1,0)

The corresponding trace catalogs are not identical.

## Main difference

### State (0,0,1)
- (A,O,O,O) is odd
- (A,E1,E1,E1) is even
- (A,E2,E2,E2) is even

### State (1,1,0)
- (A,O,O,O) is odd
- (A,E1,E1,E1) is even
- (A,E2,E2,E2) is odd

So the bridge signature does not determine which anchored even sheet carries odd activation.

## Strongest reading

The two invariant preimages differ by an anchored even-sheet polarity swap.

Therefore the bridge signature forgets more than the invariant cube:
it also forgets which anchored continuation sheet is active within a shared quotient class.
