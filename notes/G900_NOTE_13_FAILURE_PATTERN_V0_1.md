# G900 Note 13 — Failure Pattern v0.1

## What the failures have in common
The simple macro rules repeatedly produce:
- some or all of the vertical/rung-type edges
- twisted cross-couplings
- but not the top and bottom face triangles

## Meaning
The 3x2 prism support is not being lost accidentally.
It is systematically not present under naive single-cell collapse.

## Interpretation
The true quotient mechanism is likely not a one-step cell-label map.
