# Chart-Relative vs Physical Covariance Note

## Discovery

The current mixed reopening law produces:

- 011: left -> e_right, right -> e_left
- 111: left -> e_right, right -> e_left

So in literal physical edge labels, the two mixed states are identical.

## Important distinction

This does not by itself prove that the law is chart-blind in the deeper geometric sense.

Why:

For A=1, the controller uses chart-relative notions:
- chart-left
- chart-right

and these are not the same as the literal edge labels `e_left` and `e_right`.

## Correct reading

The current probe tests physical-edge covariance only.

It does not settle chart-relative covariance.

## Real semantic fork

The open question is now:

Should mixed reopening be judged by:
- literal physical edge labels, or
- chart-relative local frame coordinates?

That is the correct next derivation question.
