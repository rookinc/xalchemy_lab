# G60 Two-Face Snap Experiment

## Goal

Test the hypothesis that coordinated face perturbations produce an interior bias which is then resolved by nearest-vertex snap in G60.

## Core rule

1. one or more faces are perturbed
2. those perturbations combine into an interior reduced state
3. the reduced state snaps to the nearest G60 vertex
4. that snapped vertex becomes the next discrete readout / carrier site

## Strongest sentence

Boundary perturbations reduce in the interior and then resolve discretely by nearest-vertex snap.

## First experiment class

Use exactly two simultaneous face perturbations.

This is the smallest nontrivial case beyond the single-face regime.

## Candidate pairs

Start with simple anchor-distinct cases:

- north + east
- north + south
- east + west

These should probe:

- skew reinforcement
- opposition / cancellation
- axial bias

## What to record

For each two-face perturbation pair:

- active face perturbations
- qualitative interior bias direction
- nearest snapped G60 vertex
- resulting next carrier/readout site
- resulting candidate closure redirection

## Minimal outcome classes

The snapped result can be classified as:

- reinforced
- opposed
- neutralized
- skewed

## Suggested qualitative expectations

### north + east
- likely skewed interior bias
- likely snap toward a vertex coherent with the north-east sector

### north + south
- likely axial opposition
- possible cancellation or center-biased ambiguity

### east + west
- likely axial opposition
- possible cancellation or a different symmetry class than north + south

## Why this matters

This is the first experiment that joins:

- boundary perturbation
- interior reduction
- discrete G60 resolution

It is the first real test of a boundary-to-interior transport law.
