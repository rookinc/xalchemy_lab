# State (0,0,1) -> (0,1,1) Transition Note

## Checkpoint

The state (0,1,1) was realized by flipping e02 from the baseline state (0,0,1).

## Main effect

The anchored odd trace

- (A,O,O,O)

remains odd.

So sigma does not control the anchored odd generator.

## Activated odd traces

The following baseline-even traces become odd:

- (A,E1,E1,E1)
- (E1,E1,E1,E2)
- (E1,E1,E1,O)

So turning on sigma activates the E1-side continuation sector.

## Stable odd classes

The following remain odd:

- (A,O,O,O)
- (D,E1,E1,E2)
- (D,E1,E1,M+)
- (D,E1,E2,E2)
- (E2,E2,E2,E2)

## Strongest reading

The sigma bit is best interpreted as E1-sheet activation, while the tau bit is best interpreted as anchor-side odd exchange activation.
