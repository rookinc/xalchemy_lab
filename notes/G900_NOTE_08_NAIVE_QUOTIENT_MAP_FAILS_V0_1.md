# G900 Note 08 — Naive Quotient Map Fails v0.1

## Candidate rule tested
- prism macro = layer mod 3
- prism bit = orientation bit

## Result
This rule does produce six nontrivial quotient classes, but it does **not** recover the triangular prism support graph.

## Meaning
The carrier-derived 10-layer band, mod-3 macro phase, and 2-way bit split are real.
But the prism macro label is not given by layer phase alone.

## Conclusion
The naive carrier quotient is too coarse for prism face structure.
