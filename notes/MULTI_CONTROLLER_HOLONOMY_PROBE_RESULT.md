# Multi-Controller Holonomy Probe Result

## Result

Three multi-controller loops were compared:

- loop_A: all controllers with A=0, sigma=1, tau=1
- loop_B: all controllers with A=1, sigma=1, tau=1
- loop_C: mixed chart assignment

All three loops produced the same physical trace:

- e_right
- e_left
- e_right

But they produced different chart-coordinate traces:

- loop_A: chart_right, chart_left, chart_right
- loop_B: chart_left, chart_right, chart_left
- loop_C: chart_right, chart_right, chart_right

## Strongest conclusion

The same physical transport trace can carry different framed meanings depending on the chart assignments encountered along the loop.

Therefore the operative transport content is not exhausted by raw edge labels.

## Strongest sentence

Multi-controller loops can preserve physical exit traces while producing different chart-coordinate traces; therefore framed transport meaning accumulates across the loop.

## Interpretation

This strengthens the local holonomic-state hypothesis.

The controller register `(A, sigma, tau)` is acting like a local framed transport state, and chart assignment contributes to transport meaning across multiple sites.

## Status

This is not yet a full holonomy theory, but it is a real receipt that the runtime is holonomic in flavor rather than purely edge-labeled.
