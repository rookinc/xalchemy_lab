# Kernel as Reconciliation

## Result

The kernel should not be understood as localization alone.

It is better understood as localization plus reconciliation.

## Interpretation

The transition `r0 -> r1` must determine:

- where the spark resolves
- how the centroid is oriented to it
- how the local spark orientation and cube orientation are reconciled
- which initial grammar state is thereby seeded

## Strongest sentence

The kernel is the map from raw perturbation to reconciled centroid-oriented initial state.

## Consequence

This suggests that the emerging `Z2 x Z2` structure is at least partly kernel-resolved, rather than purely downstream.
