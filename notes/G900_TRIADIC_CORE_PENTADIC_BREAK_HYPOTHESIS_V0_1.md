# G900 Triadic Core / Pentadic Break Hypothesis v0.1

## Status
Hypothesis note.

## Core idea
Current evidence suggests that the centered symmetry belongs to the triadic core of the quotient, while the offset magnitude 5 belongs to a later propagation or extension law.

In short:

- the 3-core is centered,
- the 5-extension is where symmetry first breaks or is burdened.

## Motivation
The current checked G900 package establishes a clean centered prism law:

- top = 140
- rung = 145
- bottom = 150

so that

- 145 = (140 + 150) / 2
- 290 = 140 + 150 = 2 * 145

This is naturally a triadic picture:

- upper layer
- center layer
- lower layer

or, in centered form,

- (I - d, I, I + d)

with the present extracted instance

- (I - 5, I, I + 5).

## What seems structurally strong
The centered triadic core appears easier to force than the offset magnitude.

The present tests indicate:

- arithmetic alone does not force midpoint balance,
- centered-offset structure forces the center,
- export normalization fixes d = 5,
- but d = 5 is not yet structurally derived.

This suggests that:

- the center belongs to the core quotient structure,
- the offset magnitude belongs to a further normalization or propagation layer.

## Hypothesis
The quotient has two regimes:

### 1. Triadic core regime
This regime is governed by centered balance:

- midpoint law,
- doubled-center law,
- face-sum law,
- centered quotient reading Q(I) = 2I.

This is the clean symmetric core.

### 2. Pentadic extension regime
This regime governs how the centered core is embedded, normalized, or propagated outward.

The offset magnitude 5 may belong to this regime.
If so, 5 is not the core symmetry itself.
It is the first outer deformation, shell-step, or controlled symmetry break of that core.

## Informal slogan
Symmetry is clean at 3.
The first break appears at 5.

Or more compactly:

- 3 = centered core
- 5 = broken extension

## Why this matters
This separates two questions that should not be conflated:

1. Why is the quotient centered at 145?
2. Why is the centered split specifically ±5?

The current evidence suggests different answers:

- 145 may come from the triadic quotient core,
- ±5 may come from the pentadic propagation law.

## Candidate mathematical form
A possible future decomposition is:

- core law: Q(I) = 2I
- extension law: d = Phi(core, shell, parity, transport, normalization)

where the present extracted artifact gives d = 5.

## What would support the hypothesis
The hypothesis gains support if one can show:

- triadic centered balance survives independently of the specific value 5,
- the value 5 appears only when layer counts, shell geometry, parity transport, or normalization are added,
- or the first explicit asymmetry / burden / propagation step is pentadic in character.

## Present conclusion
The checked bundle supports a centered triadic quotient picture.
The specific offset magnitude 5 is real, but may belong to a later extension law rather than to the core itself.

So the working hypothesis is:

- the triadic core is symmetric,
- the pentadic layer carries the first controlled symmetry break.
