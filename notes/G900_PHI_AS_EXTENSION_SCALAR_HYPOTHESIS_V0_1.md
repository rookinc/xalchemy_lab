# G900 Phi as Extension Scalar Hypothesis v0.1

## Status
Hypothesis note.

## Proposal
The scalar governing the G900 extension layer may be phi rather than the previously tested cubic root.

This does **not** mean phi replaces the core quotient law.
The current strongest core law remains:

- Q(I) = 2I

with the centered prism family

- (I - d, I, I + d).

Instead, the proposal is that phi may govern the **extension geometry** that selects the shell or band structure in which d appears.

## Core vs extension split
The current checked package supports the following distinction:

### Core quotient law
- center I is structurally meaningful,
- the quotient image is 2I,
- the base quotient image is insensitive to d.

### Extension law
- d is not fixed by the core quotient image,
- d is currently explained best by the centered band picture,
- and the deeper mechanism selecting d remains open.

The phi hypothesis concerns this second layer.

## Why phi is plausible
Phi belongs naturally to the geometric family of:

- pentagons,
- 5-fold symmetry,
- shell growth,
- dodecahedral or pentadic extension patterns.

This makes phi a better candidate for the extension layer than the previously tested cubic root, which did not align with the observed G900 normalized quantities.

## Working interpretation
The current working picture is:

- the triadic core governs the centered quotient law,
- the pentadic extension governs the outer shell or band law,
- phi may be the scalar that controls that extension geometry.

In slogan form:

- core law: Q(I) = 2I
- extension scalar: phi

## What this does not yet claim
This note does **not** claim that phi has already been derived from the current exported G900 data.

It does not claim:

- I = phi,
- d = phi,
- 290 = phi-related directly,
- or that the current prism weights are immediate functions of phi.

The proposed role of phi is subtler:

- phi may govern shell ratios, extension geometry, or the pentadic layer that indirectly selects the band law.

## Strong candidate form
A plausible future decomposition is:

- core quotient law: Q(I) = 2I
- centered band law: d = L / 2
- extension selector: L or shell geometry governed by phi

So phi would not be the quotient itself.
It would be the scalar shaping the extension regime in which the quotient sits.

## What would count as evidence
The phi hypothesis gains support if one finds:

1. a clean phi-ratio in shell radii or extension distances,
2. a pentagonal or dodecahedral upstream construction whose normalization naturally introduces phi,
3. or a 5-fold carrier geometry that selects the observed extension law in a phi-governed way.

## Current conclusion
Phi is a plausible extension scalar for the 5-side geometry.
It is not yet a proved control parameter of the current G900 band law.

## Honest summary
The present package supports a discrete centered quotient core.
Phi is a live hypothesis for the extension geometry, not yet an established explanation.
