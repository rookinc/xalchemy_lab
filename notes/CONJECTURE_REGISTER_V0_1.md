# Conjecture Register v0.1

## Purpose

This note converts live speculation into structured conjectures.

Each conjecture should state:

- domain
- claim
- evidence that would support it
- failure condition that would falsify it

This is the bridge from exploration to research.

---

## Conjecture 1: Kernel primacy

### Domain
The local causal-cell / G60 kernel model.

### Claim
The unique nontrivial step of the system is the kernel transition:

- `r0 -> r1`

After `r1`, propagation is mostly deterministic grammar.

### Evidence would be
A finite downstream grammar can be given once `r1` is known, and no equally hard state-selection step reappears after `r1`.

### Failure would be
A later stage after `r1` requires a second equally nontrivial open-ended localization or reconciliation process.

## Conjecture 2: Spark primitive

### Domain
The formal input layer.

### Claim
A spark is the minimal perturbation event sufficient to localize an initial active site, orient the centroid, and seed the first grammar state.

### Evidence would be
A finite spark specification is enough to determine the kernel output up to a small finite ambiguity.

### Failure would be
Additional independent data are always needed beyond the spark to define the kernel input meaningfully.

## Conjecture 4: Lift+twist half-step

### Domain
The internal kernel microstructure.

### Claim
Lift+twist is a half-step reconciliation operator:

- `r0 -> r0.5 -> r1`

and not merely a vertex label.

### Evidence would be
A useful intermediate identification state `I` can be defined as the output of lift+twist acting on ordered `xyz`.

### Failure would be
No intermediate state is needed, or lift+twist is better modeled as a static label rather than an operator.

## Conjecture 5: Identification state I

### Domain
The reconciliation layer.

### Claim
There exists an identification state:

- `I`

produced by the lift+twist half-step, which mediates between ordered `xyz` and reconciled `r1`.

### Evidence would be
A finite rule can be written from `xyz` to `I`, and from `I` to `r1`.

### Failure would be
Either `I` collapses trivially into `r1`, or no distinct intermediate role can be assigned to it.

- `phi^3` as ignition / radiative scale
