# Signed Framed Displacement Result Template

## Goal

Compute a second framed invariant that distinguishes loops inside the same cocycle class.

## Definition

Assign:

- `chart_left  = -1`
- `chart_right = +1`

Then for a chart trace:

- `c_1, ..., c_n`

define the signed framed displacement:

- `S = sum of chart values`

## Interpretation

- positive `S` means right-biased framed displacement
- negative `S` means left-biased framed displacement
- larger magnitude means stronger net framed displacement

## What to inspect

For each loop:
- chart trace
- chart values
- `S`

## Strongest sentence

Signed framed displacement distinguishes framed traces that share the same cocycle invariant.
