Native local failure decomposition

Any local failure could only come from one of three sources:

Failure type 1. Scaffold failure

The visible branch leaves the rigid scaffold form.

That would mean one of

W,X,Y,Z,I

But the slot-4 machine is defined precisely as the scaffold-preserving family, so this is ruled out inside the local regime.

Failure type 2. Socket failure

The visible branch leaves the slot-4 socket regime.

That would mean the machine no longer has the form

[o4,s0,t0,s2,x,s4].

But the local seam theory already isolates the slot-4 branch family as the machine under study, and retained one-edit dynamics preserve that family structure.

So this is also ruled out inside the local regime.

Failure type 3. Hidden sheet failure

The visible shape survives, but the lifted branch cannot be propagated consistently under the hidden sign-sheet law.

This is the only remaining failure mode.

That is the reduction.

Proof skeleton

We prove the theorem by elimination.

Step 1. Visible scaffold preservation removes scaffold failure

By hypothesis, admissible one-edit local evolution preserves the visible slot-4 scaffold/socket form. In particular, the scaffold register

(W,X,Y,Z,I)=(o4,s0,t0,s2,s4)

Therefore failure cannot be attributed to scaffold drift.

So failure type 1 is eliminated.

Step 2. Visible socket preservation removes socket failure

The slot-4 machine has a fixed socket position , with only the payload varying. Since the visible local evolution remains within the slot-4 scaffold/socket form, the machine does not lose the socket sector itself.

Therefore failure cannot be attributed to socket loss or socket migration.

So failure type 2 is eliminated.

Step 3. Any remaining failure must occur upstairs

After Steps 1 and 2, the visible state still lies in the same scaffold/socket machine shape. So any obstruction not visible downstairs must belong to the lifted layer.

The only non-visible state component in the minimal lifted machine is the hidden sheet/sign coordinate.

Therefore any remaining local failure must be encoded as incompatibility in the hidden sheet evolution.

So failure type 3 is the only surviving one.

Step 4. Hidden sheet incompatibility is sufficient for failure

If the hidden sheet coordinate cannot be propagated compatibly with

n_{15}=-n_0,\qquad n_{30}=n_0,

Hence hidden sheet incompatibility produces local failure.

Combining Steps 1–4, local failure is equivalent to hidden sheet incompatibility once scaffold and socket preservation are fixed. ∎
