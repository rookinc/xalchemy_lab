# Local Holonomic State Hypothesis

## Hypothesis

The controller register `(A, sigma, tau)` is not merely a convenient bit tuple.
It is the smallest currently realized local framed transport state.

## Reason

The routing law is not fully intelligible in raw edge labels alone.

It depends on:
- local chart orientation `A`
- chart-relative structural bits `sigma, tau`
- transient yield memory under burden

This means the law lives in a framed local state, not just in an edge-labeled graph.

## Strongest sentence

What appears identical in physical exits may be oppositely oriented in chart coordinates, so the operative local object is a framed state rather than a raw edge choice.

## Testable consequence

If `(A, sigma, tau)` is truly a local holonomic state, then transporting through a small closed local circuit should reveal frame-sensitive return behavior not visible in bare edge labels alone.
