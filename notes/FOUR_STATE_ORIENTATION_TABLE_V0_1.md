# Four-State Orientation Table v0.1

## States

The current sketch is modeled by four canonical orientation states:

- `s0`
- `s1`
- `s2`
- `s3`

## Shared invariants

All four states preserve:

- outer loop / shell
- anchored top contact
- internal triangular frame
- central spiral/core motif
- dominant diagonal support line

## Varying features

Across the four states, the following vary:

- orientation of the internal frame
- lean of the dominant diagonal
- presentation of the inner spiral relative to the frame

## First transition rule

The first candidate transition grammar is cyclic:

- `s0 -> s1`
- `s1 -> s2`
- `s2 -> s3`
- `s3 -> s0`

## Strongest sentence

The local object is modeled as one motif occupying four cyclic orientation states.
