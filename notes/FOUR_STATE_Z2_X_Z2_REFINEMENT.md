# Four-State Z2 x Z2 Refinement

## Result

The four-state sketch family is better modeled as a product of two binary operations than as a pure cyclic rotation orbit.

## Observed rule

- rows mirror rotation
- columns flip through the diameter
- diagonals apply both operations

## State model

Model the four states as:

- `(0,0)`
- `(1,0)`
- `(0,1)`
- `(1,1)`

with two binary generators:

- `m` = mirror-rotation bit
- `d` = diameter-flip bit

## Transition meaning

- row move toggles `m`
- column move toggles `d`
- diagonal move toggles both

## Strongest sentence

The four-state family is a `Z2 x Z2` state space generated by mirror-rotation and diameter-flip.
