# G900 prism result note

The order-30 triangular subdivision carrier with 900 smallest triangular cells admits a 6-class quotient by `(macro, bit)`.

Class nodes:
- (0,0), (1,0), (2,0)
- (0,1), (1,1), (2,1)

The unweighted support graph on these six classes is exactly the triangular prism.

Weighted adjacency:
- bit-0 triangle edges: 140
- vertical macro-pair rung edges: 145
- bit-1 triangle edges: 150

So the first coarse quotient support of the G900 carrier is an exactly realized weighted triangular prism.

This is the first small finite support graph extracted from the G900 candidate carrier.
