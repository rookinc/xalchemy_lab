# Theorem: Two-Phase Anchored Controller Over a Rigid Distal Backbone

## Theorem

\[
\text{local law}

---

## 1. Two anchored phases

The anchored phase is selected by the control bit `A`.

### Phase A = 0

The anchor operates in an odd/E1 control phase:

- `(A,O,O,O)` tracks `tau`
- `(A,E1,E1,E1)` tracks `sigma`
- `(A,E2,E2,E2)` stays even

Mixed anchored traces satisfy:

- `(E1,E1,E1,O)` is odd exactly when `sigma = tau`
- `(E1,E1,E1,E2)` tracks `sigma`

So the live anchored controls are:
- odd branch activation,
- E1-sheet activation.

### Phase A = 1

The anchor operates in an E2-dominant phase:

- `(A,E2,E2,E2)` is always odd
- `(A,E1,E1,E1)` is odd except at `(sigma,tau)=(1,0)`
- `(A,O,O,O)` is odd except at `(sigma,tau)=(0,1)`

Mixed anchored traces satisfy the same shuttle law:

- `(E1,E1,E1,O)` is odd exactly when `sigma = tau`
- `(E1,E1,E1,E2)` tracks `sigma`

So the A=1 phase is not a simple sheet swap.
It is a distinct anchored control chart.

## 2. Rigid distal backbone

Across the full realized local cube, the following classes remain odd:

- `(D,E1,E1,E2)`
- `(D,E1,E1,M+)`
- `(D,E1,E2,E2)`
- `(E2,E2,E2,E2)`

And the following remain even:

- `(D,E1,O,M+)`
- `(E1,E1,M+,M+)`
- `(E1,E2,E2,E2)`
- `(E1,E2,M+,M+)`
- `(E1,M+,M+,O)`
- `(E2,E2,E2,O)`

Thus the distal transport law is rigid over the tested local cube.

## 3. Consequence

The local invariant cube `(A,sigma,tau)` is not best interpreted as three symmetric abstract bits.

Instead:

- `A` selects the anchored control phase,
- `sigma` controls E1-sheet activation,
- `tau` controls odd-branch activation,

subject to the phase selected by `A`.

So the anchored law is charted, while the distal backbone is fixed.
