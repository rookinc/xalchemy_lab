# r0 -> r1 Localization Split

## Principle

The main nontrivial transition is:

- `r0 -> r1`

After `r1` is established, propagation appears largely deterministic.

## Interpretation

### r0
Raw perturbation state before resolved localization.

### r1
First resolved state:
- perturbation location identified
- centroid oriented to the perturbation
- initial attractor/readout state set

## Consequence

The system separates into two regimes:

1. localization regime
   - `r0 -> r1`

2. propagation regime
   - deterministic or near-deterministic grammar after `r1`

## Strongest sentence

The hard step is `r0 -> r1`; after that, propagation is mostly grammar.
