# Holonomic Loop Probe Result Template

## Goal

Compare two controllers with the same mixed-state physical exit pattern but different chart assignments:

- `011`
- `111`

## What to inspect

For each step:
- handedness
- physical exit
- chart-coordinate exit
- switch state

## What would support the hypothesis

The local holonomic-state hypothesis gains support if:

- the physical exit pattern can match or nearly match,
- while the chart-coordinate readout differs systematically,
- and the controller summaries retain chart-sensitive meaning.

## Strongest sentence

If two controllers can look the same in raw edge labels but differ systematically in chart-coordinate transport, then the operative local object is framed and holonomic in flavor.
