# G15 -> G30 -> G60 Cube Lens

## Goal

Define a projection lens that sends local G15 packet data through G30 organization onto the 6 faces of a G60 cube.

## Layers

### G15
Local packet grammar:
- anchored controller
- odd exchange branch
- E1 continuation sheet
- E2 continuation sheet
- rigid distal skeleton
- cancellation channel

### G30
Transport organization layer:
- anchor axis
- branch channel
- sheet pair
- rigid scaffold
- seam law

### G60 cube
6-face display carrier.

## Proposed face deployment

- F1 = anchor controller face
- F2 = anchored E1 face
- F3 = anchored E2 face
- F4 = distal odd backbone face
- F5 = distal even backbone face
- F6 = seam / cancellation face

## Strongest reading

The cube is not merely a container.
It is the visible carrier of a packet-and-skeleton transport lens:

G15 -> G30 -> Faces(G60)
