## Goal

Test whether framed meaning admits a return class at closure, beyond raw physical traces.

## Inspect

For each loop:

- physical trace
- chart trace
- initial frame
- final frame
- return class

## Minimal classes

- `same`
- `reversed`

- `ambiguous`

## What would support the framed-holonomic reading

Support increases if loops with comparable physical traces exhibit different return classes in chart-relative coordinates.
