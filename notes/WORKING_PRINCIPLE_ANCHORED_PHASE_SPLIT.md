# Working Principle: Anchored Phase Split

## Claim

The chart split between A=0 and A=1 is best interpreted as an anchored phase split, not as a simple sheet swap.

## A = 0 phase

The anchor packet operates in an odd/E1 control phase:

- (A,O,O,O) tracks tau
- (A,E1,E1,E1) tracks sigma
- (A,E2,E2,E2) stays even

So the live anchored controls are:
- odd branch
- E1 sheet

## A = 1 phase

The anchor packet operates in an E2-dominant phase:

- (A,E2,E2,E2) is always odd
- (A,E1,E1,E1) depends on the local control pair (sigma,tau)
- (A,O,O,O) depends on the local control pair (sigma,tau)

So E2 is locked on, while odd and E1 become complementary anchored controls.

## Strongest current reading

A does not merely swap E1 and E2.
It selects the anchored phase in which local control is organized.
