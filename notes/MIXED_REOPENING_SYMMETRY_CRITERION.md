# Mixed Reopening Symmetry Criterion

## Question

Should the mixed reopened state respect chart reversal?

We compare two laws:

- M1: chart-blind mixed reopening
- M2: chart-sensitive mixed reopening

## Criterion

Chart covariance criterion:

Toggling A should act as a local frame reversal on every structurally defined routing pattern, including mixed reopening.

## Key test pair

- 011
- 111

### M1
- 011 = left->e_right, right->e_left
- 111 = left->e_right, right->e_left

So mixed reopening ignores chart reversal.

### M2
- 011 = left->e_right, right->e_left
- 111 = left->e_left, right->e_right

So mixed reopening respects chart reversal.

## Verdict under this criterion

- M1 fails chart covariance at the mixed state
- M2 satisfies chart covariance

## Interpretation

If A is only a unary/base-frame selector, M1 may still be operationally acceptable.

If A is a genuine chart involution on the full local routing law, M2 is the better form.
