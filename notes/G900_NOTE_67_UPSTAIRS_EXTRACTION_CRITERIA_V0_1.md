# G900 Note 67 — Upstairs Extraction Criteria v0.1

## A useful upstairs artifact should contain at least one of:
- named finite states
- explicit transition arrows
- admissible move list
- return / reset / closure step
- root / neutral state

## Best possible artifact
A tiny transition table or state diagram.

## Good enough artifact
A paragraph or code snippet that explicitly defines local admissible moves.

## Not enough by itself
- broad geometric prose
- general dodecahedral discussion
- high-level interpretation without local rules
