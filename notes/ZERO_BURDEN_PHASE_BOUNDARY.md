# Zero-Burden Phase Boundary
## Sign coherence first, mismatch second, stress only on the coherent floor

This note records the present status of the burden-phase-map exploration in the 4-hub patch toy.

It is a working interpretation note.
It is not a physics claim.
It is a local rule note for the present closure machinery.

The key result is now sharp enough to say plainly:

Mixed-sign triads do not enter a stress-sensitive regime.
They are already burden-positive before stress is considered.

That means the only place where a stress gate can matter is the zero-burden coherent manifold.

This is the present phase-boundary result.

---

# 1. Question being tested

The recent probes established that:

- closure class is presently burden-gated
- burden comes from sign incoherence and prior mismatch
- stress is cargo under the current law
- a future stress gate, if introduced, would most naturally be an overload threshold

The next question was therefore:

Where could a stress gate actually change the answer?

In particular:

- can stress matter anywhere in mixed-sign space?
- or does mixed sign already force tension before stress enters the story?

This note records the answer.

---

# 2. The relevant comparison

Two laws were conceptually compared.

## Current law

tension iff burden > 0

## Shadow overload law

tension iff burden > 0 or total_stress >= k

for some high threshold k.

The important point is that the shadow law only differs from the current law when:

- the current burden is zero
- but total stress is high enough to trigger overload

So the comparison is really a search for states with zero burden but high stress.

---

# 3. What the phase-map sweep showed

The sweep separated the state space into:

- coherent sign patterns
- mixed sign patterns
- varying mismatch
- varying stress

The result was very clear.

## Mixed-sign region

For the mixed-sign sector sampled, every case was already burden-positive.

That means:

- the current law returned tension
- the shadow overload law also returned tension
- the reason remained burden, not overload

So mixed-sign states do not reveal a hidden stress boundary.

They are already beyond the clean-closure phase.

## Coherent region

Only the coherent region can host zero-burden states.

That means only the coherent region can meaningfully test whether stress overload would create a new boundary beyond the current law.

This is the decisive structural result.

---

# 4. Main interpretation

The emerging closure logic now looks lexicographic.

## First gate: sign coherence

If the incoming triad is sign-incoherent relative to the local hub polarity, burden is already positive.

So tension is already selected.

## Second gate: prior mismatch

Even if signs are coherent, nonzero mismatch also makes burden positive.

Again, tension is already selected.

## Third gate: possible overload

Only after both of the above vanish does there exist a clean state that stress could potentially destabilize.

So stress can only matter on the coherent zero-burden floor.

That is the current best reading of the toy.

---

# 5. Zero-burden manifold

The phrase zero-burden manifold is useful here.

It means the set of incoming triads satisfying:

- sign coherent with the hub
- zero prior mismatch

For a positive hub, that means the cleanest cases are of the form:

- +,+,+
- mismatch 0,0,0

For a negative hub, the analogous clean cases are:

- -,-,-
- mismatch 0,0,0

Only these states lie on the zero-burden manifold.

Everything else is already burden-positive.

So if a future overload law is added, its true action is not global.
Its action is restricted to this narrow manifold.

That is a strong simplification.

---

# 6. What this excludes

This phase-boundary result excludes a tempting but incorrect interpretation:

stress is not a universal competing variable everywhere in state space.

In particular, the model does not currently suggest that:

- mixed-sign states can become clean if stress is low
- mixed-sign states need stress to become tensioned
- stress can "rescue" incoherent sign configurations

None of that is supported.

Mixed sign is already enough to create burden.
Mismatch is already enough to create burden.
Stress is therefore downstream of those gates, not prior to them.

---

# 7. Best current local law

The cleanest current formulation is:

Triadic closure is burden-gated first.
Only coherent zero-burden states can meaningfully test a future overload gate.

A slightly more expanded version is:

Sign defect and mismatch determine ordinary tension.
Stress, if later promoted into classification, would act only as an overload mechanism on otherwise coherent clean triads.

That is a very good local law statement.

---

# 8. Why this matters

This gives the toy a more organized architecture.

Instead of one undifferentiated "energy" notion, the model now appears to contain:

- polarity structure
- burden memory
- cargo load

These are not doing the same job.

## Polarity structure

Determines sign coherence or defect.

## Burden memory

Determines whether closure is already tensioned.

## Cargo load

May matter dynamically, and may matter classificatorily only in a future overload regime.

This is an unexpectedly clean decomposition.

---

# 9. What the next mathematical target is

The next sensible target is now obvious:

map the zero-burden phase boundary directly

That means restricting attention to:

- coherent sign states only
- mismatch exactly zero
- varying stress

Then compare:

- current law
- overload shadow law

That would produce the real candidate overload boundary:

- coherent clean
- coherent overloaded
- burden tension

This is much cleaner than scanning the whole state space at once.

---

# 10. Suggested next partition of closure classes

If the overload idea is pursued further, the clean partition would be:

- closed
- tension_burden
- tension_overload

In present language, that means:

## Closed

Coherent sign, zero mismatch, stress below overload threshold.

## Burden tension

Any sign defect or mismatch burden.

## Overload tension

Coherent zero-burden triad whose aggregate stress exceeds a chosen threshold.

That partition preserves the base law while allowing a disciplined extension.

---

# 11. Best compact summary

The shortest faithful summary is:

Mixed sign never reaches the stress question.
Stress only matters on the zero-burden coherent floor.

A slightly more formal version is:

The candidate overload boundary is not a global phase boundary.
It is a boundary internal to the coherent zero-burden manifold.

That is probably the most important sentence in this note.

---

# 12. Bottom line

The phase-map exploration sharpened the closure story considerably.

What is now established is:

- mixed-sign triads are already burden-positive
- mismatch-positive coherent triads are already burden-positive
- therefore both are already in the tension regime before stress is considered
- the only states where a stress gate could matter are coherent zero-burden states
- so a future overload law would act only on the zero-burden manifold, not across the whole state space

The most important conclusion is:

Sign coherence is the first gate.
Mismatch is the second gate.
Stress only becomes a meaningful classifier on the coherent zero-burden manifold.

That is the present phase-boundary result.
