# G900 triangle descent result

The first coarse `(macro, bit)` quotient of the G900 carrier is an exact weighted triangular prism.

Collapsing the three macro rungs yields a 3-node quotient triangle.

Results:

## Coarse rung-collapsed quotient
- (0,1) = 290
- (0,2) = 290
- (1,2) = 290

So the coarse quotient is an exact uniformly weighted triangle.

## Even parity slice
- (0,1) = 160
- (0,2) = 160
- (1,2) = 160

So the even slice also descends to an exact uniformly weighted triangle.

## Odd parity slice
- (0,1) = 130
- (0,2) = 130
- (1,2) = 130

So the odd slice also descends to an exact uniformly weighted triangle.

## Interpretation

The macro quotient is perfectly isotropic.

The prism level records the binary split between sheets.
The triangle level forgets that split and retains only exact triadic symmetry.

Thus the descent observed so far is:

- sampled triangular carrier
- weighted prism
- exact weighted triangle

with parity-refined triangle weights:
- even = 160
- odd  = 130
- coarse = 290 = 160 + 130
