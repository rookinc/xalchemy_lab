# Toy G60 v1 Full-Cube Failure Note

## Checkpoint

Toy G60 Automaton v1 fit the 5-state neighborhood exactly, but it does not extend unchanged to the full realized invariant cube.

## Main failure

The distal backbone remains stable, but the anchored control surface changes behavior in the A=1 sector.

In particular, the states

- (1,0,1)
- (1,0,0)
- (1,1,1)

show anchored parity patterns not captured by the v1 rule
" A swaps anchored sheets and rearms odd ".

## Strongest reading

The A-control is not a simple anchored swap.
It acts as a higher-order mode selector on the anchored control surface.

So the current local law appears to have at least two anchored control charts:

- an A=0 chart
- an A=1 chart

## Consequence

Toy G60 Automaton v1 should now be reinterpreted as a valid local chart on part of the state space, not as the full-cube law.

The next task is Toy G60 Automaton v2 with chart-dependent anchored rules.
