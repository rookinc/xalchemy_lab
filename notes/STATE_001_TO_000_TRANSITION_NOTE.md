# State (0,0,1) -> (0,0,0) Transition Note

## Checkpoint

The state (0,0,0) was realized by flipping e01 from the baseline state (0,0,1).

## Main effect

The anchored odd trace

- (A,O,O,O)

changes from odd to even.

So the anchored exchange generator is turned off in the transition
- (0,0,1) -> (0,0,0).

## Redistribution effect

Oddness does not disappear entirely.
Instead, new odd mixed traces appear:

- (E1,E1,E1,O)
- (E2,E2,E2,O)   [one realization]

So turning off the anchored odd generator redistributes oddness into downstream branch/sheet mixed traces.

## Stable odd classes

The following remain odd across the transition:

- (D,E1,E1,E2)
- (D,E1,E1,M+)
- (D,E1,E2,E2)
- (E2,E2,E2,E2)

So distal-defect oddness and pure E2 residue are not controlled by tau.

## Strongest reading

The tau bit is best interpreted as anchor-side odd activation, not as total odd content.
