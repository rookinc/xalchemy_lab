## Description

This repository is a working research and development archive spanning:

- local and global transport constructions
- G15 / G60 / G900 descent work
- proof-packet notes and theorem scaffolds
- browser-based visualization pages under `http/`
- Python research code under `src/`
- paper-facing artifacts under `specs/` and `notes/`

This deposit is intended as a **raw snapshot** of the repository state for archival, reproducibility, and citation.

## Nature of this archive

This is a large, direct repository dump. It may include:

- active work-in-progress material
- experimental notes
- generated artifacts
- intermediate proof scaffolds
- renderer and browser prototypes

It should be read as a research workspace snapshot rather than a polished software release.

## Repository structure

Top-level directories include:

- `src/` — Python source and research tooling
- `notes/` — theorem notes, checkpoints, and proof-packet text
- `specs/` — structured paper/app artifacts
- `http/` — lightweight browser viewers and proof pages
- `renders/` — exported visual artifacts
- `export/` — generated outputs and probes
- `config/` — local config files
